Wertgutachten-Tool: einrichten & benutzen

Einmal einrichten (ca. 15 Min.), danach pro Fall ein Befehl. Du arbeitest fast nur in einem Programm — Claude Code. Ein Terminal brauchst du nicht.

Es gibt nur zwei Orte — achte oben auf jedem Schritt auf das farbige Schild:

🌐 Browser

Nur zum einmaligen Anmelden (GitHub, Claude, API-Key) und für die Einladungs-Mail.

▣ Claude Code

Die Desktop-App. Hier passiert alles andere — Einrichtung und später jedes Gutachten.

Claude Code erledigt die Installation selbst. Du sagst ihm in normalen Worten, was es tun soll — kein Terminal, keine Programmierbefehle.

Teil 1 · Einmal anmelden

🌐 Im Browser

Drei Konten anlegen

  1. GitHub-Konto mit deiner Gmail anlegen → github.com/signup. Danach deinen Benutzernamen an Elias schicken, damit er dich einlädt.
  2. Claude Code installieren (Desktop-App) und mit deinem Claude-Konto anmelden → claude.ai/download. Die Web-Version auf claude.ai reicht nicht — es muss die App sein.
  3. API-Key erzeugen → console.anthropic.com/settings/keys → „Create Key". Die Zeichenkette (beginnt mit sk-ant-…) kopieren und kurz sichern — du brauchst sie in Teil 2, Schritt 3.

Teil 2 · Einrichten — in Claude Code

▣ In Claude Code

1Claude Code öffnen

Öffne die Claude-Code-App. Sie fragt dich nach einem Ordner zum Arbeiten — wähl z. B. deinen Dokumente-Ordner. Mehr ist hier nicht zu tun.

🌐 Im Browser

2Einladung annehmen

Öffne die E-Mail von Elias, die dich ins private Repo eingeladen hat, und klick auf „Accept invitation". Fertig — du musst dort nichts weiter tun. (Das schaltet deinen Zugang frei.)

▣ In Claude Code

3Einrichtungsprompt eingeben

Kopier diesen Text und schick ihn an Claude Code. Setz ganz hinten deinen API-Key ein. Claude erledigt Download und Einrichtung dann selbst — bestätige nur, wenn es nachfragt.

Lade das Repo https://github.com/Gutachter97/wertgutachten_v1_elias.git in diesen Ordner herunter, führe danach ./setup.sh aus und trag meinen Claude-API-Key in die Datei plugins/certa/tool/.env ein. Falls Python oder Git fehlen, installier sie bitte. Mein API-Key ist: sk-ant-DEIN-KEY-HIER

Das dauert ein paar Minuten (es lädt u. a. ~300 MB fürs PDF). Wenn Claude um Erlaubnis fragt, immer auf „Allow" klicken — und einfach durchlaufen lassen.

Nur beim allerersten Mal auf einem neuen Mac: Es kann sein, dass macOS selbst einmal ein Fenster „Befehlszeilentools installieren" zeigt (→ „Installieren" klicken) oder nach deinem Mac-Passwort fragt (→ eintippen). Das ist normal, passiert höchstens einmal, und du bleibst dabei in Claude Code. War der Mac schon mal in Benutzung, kommt das meist gar nicht.
▣ In Claude Code

4Plugin aktivieren

Zum Schluss zwei kurze Befehle — einzeln nacheinander in die Eingabezeile tippen.

/plugin marketplace add wertgutachten_v1_elias /plugin install certa@certa-skills

Kontrolle: /reload-plugins meldet 1 Plugin · 1 Skill. Dann ein neues Claude-Code-Fenster öffnen — fertig eingerichtet.

Unsicher beim Pfad? Frag Claude einfach: „Wie lautet der Pfad zum Ordner wertgutachten_v1_elias?" — und setz die Antwort beim ersten Befehl statt des Namens ein.

Teil 3 · Im Alltag — pro Fall

▣ In Claude Code

→Gutachten rechnen

Ein Befehl, der Fallordner-Pfad dahinter.

/certa:wertgutachten "<Pfad zum Fallordner>"

Claude prüft die Unterlagen, holt den Bodenrichtwert online, rechnet nach ImmoWertV 2021 (Sach-/Ertrags-/Vergleichswert) und hält bei den Entscheidungspunkten kurz für deine Freigabe an.

Was rauskommt

Bearbeitbares HTML-Wertgutachten
Zahlen und Texte passen sich live an, wenn du etwas änderst. Am Bildschirm Arbeitsfassung mit SV-Markierungen, im Druck das saubere Kunden-PDF.

PDF drucken
Aus dem HTML direkt das fertige Kunden-PDF erzeugen.

In CERTA Value hochladen
Die Daten lassen sich anschließend in CERTA Value übernehmen.
